CoreInventory - Modular IMS
A centralized, real-time application designed to replace manual registers and scattered tracking methods.

Key Modules 
Product Management: Create and update products with SKUs and categories.
Inventory Operations: Handles Receipts (Incoming), Delivery Orders (Outgoing), and Internal Transfers.
Real-time Dashboard: Live KPIs for stock levels, pending receipts, and deliveries.
Stock Adjustments: Tool to fix mismatches between physical counts and system records.

Target Users 
Inventory Managers: For overseeing high-level stock operations.
Warehouse Staff: For daily picking, packing, and shelving.
